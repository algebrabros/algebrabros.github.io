how to save gnmath as an html file

go to https://cdn.jsdelivr.net/gh/gn-math/gn-math.github.io@main/singlefile.html?t=, then do ctrl+S, then change the name of the file to anything, but make sure the end of the name is .html, not .txt or anything else, then save. have fun
